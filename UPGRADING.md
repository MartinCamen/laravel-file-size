# Upgrading

This document describes breaking changes and how to upgrade between major versions.

## From 0.x to 1.0

This is the initial stable release. If you were using a pre-release version, please review the full API documentation in the README.

### Key Points

- All methods are now finalized and follow semantic versioning
- Configuration structure is stable
- Public API is locked for 1.x releases

## Future Versions

When upgrading between major versions in the future, this document will be updated with:

- Breaking changes
- Deprecated features being removed
- Migration paths
- Code examples for updating your implementation
